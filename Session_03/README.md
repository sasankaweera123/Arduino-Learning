# ABOUT THIS PROJECT

This Project IS BASED on THE ALARMS AND BEEPS USING BUZZERS FOR SECURITY

### Functions

- tone()
- noTone()

## COMPONENTS AND SUPPLIES

- Buzzer (active/passive)
- Arduino UNO (Can Use any arduino board)
- Jumper Wires (generic)
- Resistor 100ohm (optional)
