# ABOUT THIS PROJECT

 Arduino board to control the brightness of an LED.

## COMPONENTS AND SUPPLIES

- LED (generic)
- Arduino UNO (Can Use any arduino board)
- Jumper Wires (generic)
- Resistor 220ohm
