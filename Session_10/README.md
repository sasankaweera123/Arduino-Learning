# ABOUT THIS PROJECT

This Project is Based on Relay

Connecting a relay with arduino and as an example blinking a bulb

### Functions

- digitalWrite()
- delay()

## COMPONENTS AND SUPPLIES

- Arduino UNO (Can Use any arduino board)
- Jumper Wires (generic)
- Relay (generic)
- Bulb (generic)
