# ABOUT THIS PROJECT

This Project is Based on Servo Motor.

Sweeps the shaft of a RC servo motor back and forth across 180 degrees.

### Functions

- #include <Servo.h>
- attach(pin)
- analogRead(potpin)
- write(val)

## COMPONENTS AND SUPPLIES

- Arduino UNO (Can Use any arduino board)
- Jumper Wires (generic)
- Servo Motor
