# ABOUT THIS PROJECT

This Project is Based on LDR.

Using LDR sensor and LED show the mechanism.

### Functions

- analogRead(ldrPin);
- digitalWrite()

## COMPONENTS AND SUPPLIES

- LDR , 5 Mohm
- Arduino UNO (Can Use any arduino board)
- Jumper Wires (generic)
- 10K ohm Resistor
